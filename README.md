# Python_Cosine-Similarity_data_recommendn
> pandas,scikitlearn라이브러리와 코싸인 유사도 이용 데이터 추천 프로그램

## 프로그램 설명
> 4만개 이상의 준비된 데이터를 정제 후 코싸인 유사도 함수를 만든다. 
> 유사도에 따라 영화들을 정렬하고 기준이 되는 영화를 정한 후 선택한 숫자만큼 가장 유사한 영화를 추천한다
